Lets Talk!

The MeshKeeper community is here to help you!

We really want you to get involved in the MeshKeeper and we love contributions!. One of the easiest ways to get involved is to join the community discussions…

Developer Mailing List

The developer list is used for general discussions and ideas about the code and how to use it.

To post an email: meshkeeper-dev@fusesource.org
To subscribe: meshkeeper-dev-subscribe@fusesource.org
To unsubscribe: meshkeeper-dev-unsubscribe@fusesource.org

Commit Mailing List

The commit list is notified of all changes to the source code or changes to issues so you can use this list to watch development and support occur.

To post an email: meshkeeper-commits@fusesource.org
To subscribe: meshkeeper-commits-subscribe@fusesource.org
To unsubscribe: meshkeeper-commits-unsubscribe@fusesource.org
